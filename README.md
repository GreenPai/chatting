### 프로젝트 소개

이 프로젝트는 웹 소켓(WebSocket)을 이용해 간단한 통신을 구현하는 예제입니다.  
버튼을 클릭하면 서버가 VSCode 터미널(console)에 메시지를 출력하도록 구성되어 있습니다.

---

### 주요 기능

- Express 기반 간단 서버  
- WebSocket 서버 (ws 라이브러리 사용)  
- 클라이언트에서 버튼 클릭 시 WebSocket 메시지 전송  
- 서버에서 메시지 수신 후 터미널에 로그 출력
